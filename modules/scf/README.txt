Requirements:

PHP >= 5.2.0

zlib [1.2.3]
zip [2.0.0] incl. libzip [0.7.1]
  NOTE: this is not enabled by default in MAMP
     
libxml2 [2.6.27]
libxslt [1.1.20]

Hints:

- Install/enable file uploads

- Goto admin/settings/uploads: add "zip" to permitted file extensions
  for editor, and set max sizes up to e.g. 20MB/1000MB for editor.
